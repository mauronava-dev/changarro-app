# Manuales técnicos

Guías para desarrolladores que quieran entender, contribuir o adaptar Changarro.

Estos documentos asumen familiaridad con desarrollo web, pero están escritos
para que un desarrollador de cualquier nivel pueda seguirlos.

---

## Contenido

| # | Documento | Descripción |
|---|-----------|-------------|
| 01 | [Arquitectura](./01-arquitectura.md) | Estructura general del sistema y decisiones técnicas |
| 02 | [Entorno de desarrollo](./02-entorno-desarrollo.md) | Cómo preparar tu máquina para trabajar en el proyecto |

---

## ¿Por dónde empezar?

Si es tu primera vez contribuyendo, lee primero la [Arquitectura](./01-arquitectura.md)
para entender cómo está organizado el proyecto, y luego sigue con
[Entorno de desarrollo](./02-entorno-desarrollo.md) para configurar tu máquina.
